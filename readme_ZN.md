# 🌲 圣诞树排列编辑器
![](img.png)

## 🎯 这是什么？
一个**可视化工具**，让你在正方形内排列圣诞树，**分数越低越好**。

## ✨ 核心功能
- **拖拽旋转**：直观调整树的位置和角度
- **自动评分**：实时计算排列紧凑度
- **碰撞提示**：显示树之间的重叠
- **方向键移动**：一键将树推到最远位置

## 🚀 3步上手
1. **添加树** → 2. **调整位置** → 3. **降低分数**

## 📊 分数规则
```
分数 = 包围正方形面积 ÷ 树的棵数
```
**目标：让这个数字越小越好！**

## 💡 使用技巧
- 用**方向键**快速推开树
- 观察**红色虚线框**的尺寸
- 避免树的重叠
